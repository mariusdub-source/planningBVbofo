# Planning Maritime - postes horaires + colonne total semaine

Application React avec :

- mot de passe site : `BV2026#`
- mot de passe admin : `Marius24#`
- modification uniquement en mode admin
- 20 salariés
- 7 jours visibles
- colonne Total semaine à droite de dimanche
- guichet en liste déroulante
- poste en liste déroulante sous le guichet
- heure début / fin / pause remplies automatiquement selon le poste choisi
- total journée automatique ou manuel selon le modèle horaire
- saisie manuelle
- total semaine
- total mensuel
- export CSV

## Installation

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
