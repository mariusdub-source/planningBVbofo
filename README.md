# Planning Maritime - Poste sous guichet

Cette version corrige l'écran principal :

- colonne Total semaine à droite de Dimanche
- champ Guichet
- champ Poste juste sous Guichet
- Poste = liste déroulante des postes associés aux horaires
- quand un poste est choisi, début / fin / pause / total sont remplis
- total semaine calculé par salarié
- mot de passe site : `BV2026#`
- mot de passe admin : `Marius24#`

## Installation

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
