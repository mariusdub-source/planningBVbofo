# Planning Maritime - Affichage cartes

Application React avec affichage inspiré de la capture :

- colonne gauche avec postes
- cartes à droite
- panneau de modification
- mot de passe accès : `BV2026#`
- mot de passe admin : `Marius24#`
- modification uniquement en mode admin
- export CSV
- totaux par salarié
