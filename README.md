# Planning Maritime - affichage principal simplifié

Cette version modifie l'affichage principal :

- chaque case affiche uniquement :
  - le lieu / guichet
  - le poste
- il faut cliquer sur une case pour ouvrir le détail
- le détail contient :
  - guichet
  - poste
  - début
  - fin
  - pause de
  - pause à
  - total manuel
  - note
  - total journée
- colonne Total semaine à droite de Dimanche
- mot de passe site : `BV2026#`
- mot de passe admin : `Marius24#`

## Installation

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
