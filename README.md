# Planning Maritime

Application React + Vite prête pour GitHub et Vercel.

## Installation locale

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## Paramètres Vercel

- Framework Preset : Vite
- Install Command : `npm install --no-package-lock`
- Build Command : `npm run build`
- Output Directory : `dist`
